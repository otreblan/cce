# Compilador

## Dependencias

* `asciidoctor`
* `bash-completion`
* `bison`
* `cmake`
* `libfl-dev`
* `libfmt-dev`

## Build
``` bash
git clone https://github.com/otreblan/cce
cd cce

mkdir -p build
cd build

cmake ..
make
```
